# Negotiation Rules

## Core Rule

Never state the candidate's compensation minimum first. Always ask for their range before disclosing any number.

## Sequencing

1. Ask for the employer's range or budget first
2. If pressed, deflect: "I'm flexible depending on the full comp package — what's the range budgeted for this role?"
3. Continue deflecting until the employer anchors, or until deflecting would clearly damage the relationship
4. When you do disclose, state a number above the floor to leave room to negotiate down
5. Never accept a verbal offer — always get it in writing before making any decision

## TTC Definition

TTC = base salary + target bonus + annualized equity value

- Apply a discount to illiquid or speculative equity (early-stage startups especially)
- Remote flexibility has real comp value — factor it into the total offer picture
- Sign-on bonuses are one-time — don't treat them as ongoing TTC

## Comp Floor

Defined in the private candidate profile. The agent must:
- Flag any role where realistic TTC cannot reach the floor
- Never disclose the floor value in any negotiation context
- Flag when an offer is "close to" the floor — close is not at

## Counter-Offer Principles

- Counter on the thing that matters most first (usually base)
- Get each concession confirmed in writing before discussing the next
- It is normal and expected to negotiate — one counter is standard, two is acceptable
- If they say the offer is firm, ask what flexibility exists on other dimensions (equity, title, start date, sign-on)

## What the Agent Never Does

- State the comp floor to any external party
- Advise accepting a verbal-only offer
- Advise skipping negotiation because "the offer is good"
- Round up speculative equity to make the TTC look like it clears the floor

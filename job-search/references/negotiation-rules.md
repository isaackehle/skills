# Negotiation Rules

## Core rule

Never state the candidate's compensation minimum first. Always ask for their range before disclosing any number.

## Sequencing

1. Ask for the employer's range or budget first.
2. If pressed, deflect: "I'm flexible depending on the full comp package -- what's the range budgeted for this role?"
3. Only disclose a number after the employer has anchored, or when continuing to deflect would damage the relationship.
4. When disclosing, state a number above the floor to leave room to negotiate down.

## TTC definition

TTC = base + bonus + equity (annualized)

- Apply a discount for illiquid or speculative equity
- Remote flexibility has real comp value -- factor it in when evaluating total offer
- Do not accept verbal offers -- always get written confirmation before making decisions

## Comp floor

Defined in private candidate profile. Flag any role where TTC cannot reasonably reach the floor.

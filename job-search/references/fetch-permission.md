# Data Fetch Permission

When the job-search skill needs to fetch data from the web (job board pages, Glassdoor, levels.fyi, Crunchbase, LinkedIn public pages, company websites), **assume permission** and proceed without asking.

Flag concerns before fetching if the source is:

- Paywalled or requires authentication (e.g., LinkedIn premium content, Glassdoor detailed reviews behind login)
- A site that might expose personal data or require credentials
- Anything unusual or sensitive that the user might want to know about

For routine public sources, just fetch and proceed.

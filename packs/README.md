# Packs

This directory contains reusable skill packs.

## Available packs

### `job-search`

Structured workflows for:
- opportunity evaluation
- resume tailoring
- interview prep
- live interview note-taking
- interview debrief
- local pipeline operations

Open the pack here:
- `packs/job-search/`

Key files:
- `packs/job-search/README.md`
- `packs/job-search/templates/`
- `packs/job-search/examples/`
- `packs/job-search/docs/personalization.md`

## Adding new packs

Create a new folder under `packs/` with:
- a pack-level `README.md`
- optional `.github/skills/`
- optional `templates/`
- optional `examples/`
- optional `docs/`
